An **interactive Excel dashboard** designed to analyze **hotel and resort booking and cancellation patterns** across customer types.  
This project helps operations managers make **data-driven decisions** by visualizing key trends in cancellations and bookings.  

---

## 📊 Dashboard Overview
- **Customer Segments:** Couples, Families, Singles  
- **Metrics Tracked:** Bookings, Cancellations, Total Trends  
- **Features:**
  - Interactive slicers & filters  
  - Clean data visualizations (bar charts, pie charts, KPIs)  
  - Summary metrics (e.g., 32.4K cancellations by couples)  
  - Insights for operational planning  

---

## 🎯 Purpose
The dashboard provides a **quick snapshot of booking behaviors** and cancellation trends, enabling hotel and resort managers to:  
- Identify which customer groups cancel most often  
- Optimize marketing and retention strategies  
- Improve occupancy forecasting  
- Enhance overall decision-making with visual data
